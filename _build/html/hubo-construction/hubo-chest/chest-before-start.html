Computer¶

The computer has numerous parts to it. They are (in order from bottom to top):

  • The Base (constructed by KAIST and Rainbow)

  • The Hard Drive (SSD)

  • CAN Board

  • Main board with Processor on top and RAM on underside

  • Wireless adaptor (inserted in Main processing board)

  • Heat sync

  • Temperature Sensor

TEMP_SENSOR.JPG

  • Top Cover
